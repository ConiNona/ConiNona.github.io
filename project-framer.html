Project

Dear Keira:
A Digital Birthday Card

Framer Calligraphr Gimp UI / UX Personal Project
[ project screenshot ]
Overview

In late 2025, I had a dear friend's birthday coming up and I wanted to do something more meaningful than a typical birthday card. What started as a simple idea quickly turned into a fully responsive website.

"Dear Keira" is a personal project where I explored how a message can become an experience. Instead of just presenting content, I focused on creating a certain feeling through layout, typography, and pacing.

View Project →
Process
  • Brainstormed and started with a general concept: an interactive birthday message.
  • Decided on the tools I'll use to make it happen: Framer, Gimp, and Calligraphr.
  • Designed and built the site in Framer.
  • Created a custom handwritten font using Calligraphr.
  • Designed a custom logo and hand-drawn visual elements to add to the personal feel, utilizing tools like Gimp.
  • Built fully functional mobile and tablet versions, adapting layout, buttons, and interactions for each format.
  • Refined spacing, structure, and interactions to improve overall flow and consistency.
Key Elements
Fully responsive design Custom typography Personal visual identity Focus on storytelling and user experience
Reflection

This project pushed me to think beyond static design and consider how people move through something. It made me more aware of how small decisions (like spacing or timing) can change the overall experience.

What made this one special

Most portfolio projects start from a brief or a client need. This one started from wanting to make someone feel something. That shift, from 'build a site' to 'create a moment,' changed how I thought about every decision. The custom font alone took several rounds of iteration to feel right, and that attention to detail carried through everything else. It's the project I'm most personally proud of.
