Project

CMarine Surveyors:
Website Redesign for Client

WordPress SEO Responsive Design Client Project
[ project screenshot ]
Overview

"CMarine Surveyors" is a client project where I redesigned an existing website that wasn't bringing in any leads. The goal was to turn it into something clearer, more professional, and actually useful for attracting new clients.

The main challenge wasn't just design. The client didn't have a very defined structure or focus, so part of the work was figuring out how to present the services in a way that made sense and matched how people search for them.

View Project →
Process
  • Reviewed the existing site to understand what wasn't working.
  • Identified the main services, focusing on pre-purchase and insurance & finance surveys as the core of the business.
  • Reworked the structure of the site to make services clearer and easier to navigate.
  • Designed and built a more consistent layout across all pages using a WordPress template.
  • Created fully responsive versions for desktop, tablet, and mobile.
  • Built the content with SEO in mind, especially using location-based keywords.
  • Thought through how potential clients search (for example, 'marine surveyor + location') and reflected that in the structure.
  • Added credibility elements like associations, certifications, and client references.
  • Went through multiple refinements to improve flow, clarity, and overall usability.
Key Elements
Clear, service-focused structure SEO-driven content Fully responsive design Strong focus on credibility and trust Designed to generate leads, not just exist
Outcome

The final result is a much clearer and more professional website that's built to bring in potential clients, not just showcase information. It also sets a solid base to keep improving the site over time based on results.

Reflection

This project made me think more about the strategy behind a website. It's not just about how it looks, but how it works, how people find it, and how clearly it communicates what's being offered.

It also gave me the opportunity to work with WordPress, which I see as a valuable skill going forward.

Working with a real client

One of the things I didn't expect going into this project was how much of the work would happen before touching any design tool. Understanding the client's business, figuring out what they actually needed versus what they thought they needed, and translating that into a structure that works for real users. That was its own challenge. It taught me that good web work is as much about listening and thinking as it is about building.
